# CORPPAZ — Brand Identity & Landing Page Design System

# Brand Overview

Nombre:
CORPPAZ — Corporación Todos por la Paz

Posicionamiento:

CORPPAZ no es una ONG pequeña.
Es una corporación que ejecuta proyectos sociales, agropecuarios, humanitarios y de desarrollo territorial para entidades públicas y privadas.

La marca debe transmitir:

- confianza
- institucionalidad
- experiencia
- transparencia
- impacto social
- capacidad logística
- presencia en territorio
- profesionalismo

No debe verse infantil.
No debe parecer una fundación escolar.

Debe sentirse como una mezcla entre:

- Naciones Unidas
- USAID
- Cruz Roja Internacional
- Banco Mundial
- Deloitte Government
- Gobierno Digital

# Concepto Visual

El concepto principal es:

"Construimos desarrollo donde más se necesita."

La web debe sentirse como una organización que ejecuta proyectos reales en campo.

No como una organización que simplemente habla de ellos.

Las fotografías son el principal elemento visual.

No usar ilustraciones.

No usar dibujos.

No usar iconografía infantil.
